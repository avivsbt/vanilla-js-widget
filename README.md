# Recommendation Widget

## Actions

### Installation:
1. Clone the project.
2. Run `npm install`.

### Local Development:
1. Run `npm run dev`.

### Running Tests:
1. Run `npm run test`.

### Building for Production:
1. Run `npm run build`.
